Selected Research, Software & Product Work

Machine Learning Assisted Chemo-Dynamical Analysis of Milky Way Stellar Populations Using Gaia DR3 and LAMOST

An independent research-oriented project in Galactic Archaeology combining Gaia DR3 astrometry with LAMOST stellar spectroscopy to investigate the chemo-dynamical structure of Milky Way stellar populations.

Gaia DR3 · LAMOST · Galactic Archaeology · Chemo-dynamics · Stellar Populations · Machine Learning

Project details

The project includes catalogue exploration, Gaia–LAMOST cross-matching, stellar parameter validation, distance and kinematic feature construction, colour–magnitude and velocity-space visualization, and machine-learning-assisted identification of candidate stellar population structures.

View project on GitHub

OpenMed: Interactive Medical Simulation Software for Cardiovascular Intervention Training

A research-oriented medical simulation software project focused on interactive 3D visualization, procedural workflow design, and lightweight simulation interfaces for cardiovascular intervention training and experimentation.

Medical Simulation · Cardiovascular Intervention · 3D Visualization · React · Three.js · Research Software

Project details

The project develops a browser-based simulation environment with 3D anatomical visualization, dual-view interaction design, guidewire control logic, procedural state management, parameter panels, workflow tracking, and exportable interaction metrics for research-oriented medical simulation workflows.

View project on GitHub

Geo Sales Automation MVP: AI-Assisted Localized Content Generation and Distribution Workflow

An independently designed and implemented internship project focused on building a product-style MVP for AI-assisted localized sales content generation, channel-specific post creation, and lightweight distribution workflow management.

Independent Internship Project · Product MVP · AI Content Generation · Sales Automation · Geo-localized Marketing · React · Express · OpenAI API
